# 更新日志

## [1.0.0] - 2026-04-02

### 新增
- 🎉 首次发布
- ✨ 支持自动生成整体项目软著申请材料
- ✨ 支持生成独立AI模块申请材料
- ✨ 提供交互式命令行界面
- ✨ 自动扫描项目代码并生成源代码文档
- ✨ 自动生成软件著作权登记申请表
- ✨ 自动生成用户手册/设计说明书
- 📦 内置5个AI模块配置模板
- 🎨 美观的命令行输出（使用 chalk 和 ora）
- 📖 完整的使用文档和示例

### 功能特性
- 支持多种编程语言代码扫描（.js, .jsx, .ts, .tsx, .py, .java, .cpp, .c, .h, .go, .rs）
- 智能排除 node_modules、build、dist 等目录
- 自动生成前30页+后30页源代码文档
- 支持自定义输出目录
- 提供 Node.js API 供程序化调用

### 内置模块
- ai-question-generator - AI智能面试题目生成系统
- digital-video-generator - 数字人视频自动生成系统
- video-interview-system - 智能视频面试系统
- interview-report-system - AI面试报告生成系统
- task-scheduler-system - 智能任务调度系统
