## 🚀 使用方法

### 方式1：npm命令（推荐）
```bash
npm run copyright                           # 交互式生成
npm run copyright:full                      # 生成整体项目
npm run copyright:modules <模块名>          # 生成指定模块
```

### 方式2：Python脚本
```bash
python3 scripts/generate_module_copyright.py <模块名> .
```
