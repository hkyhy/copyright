## 二、系统概述

### 2.1 系统目标
基于WebRTC的智能视频面试系统，支持实时视频面试、设备检测、面试录制和流程自动化

### 2.2 技术架构
React + WebRTC + Agora/TRTC + Node.js + Fastify

### 2.3 系统特点
1. WebRTC实时视频通信
2. 智能设备检测和配置
3. 面试流程自动化引导
4. 视频录制和存储
5. 多种面试类型支持
6. 问卷调查集成
7. 面试结果实时反馈
