# 智能视频面试系统

## 软件信息
- 名称：智能视频面试系统
- 版本：V1.0
- 描述：基于WebRTC的智能视频面试系统，支持实时视频面试、设备检测、面试录制和流程自动化

## 主要功能
- WebRTC实时视频通信
- 智能设备检测和配置
- 面试流程自动化引导
- 视频录制和存储
- 多种面试类型支持
- 问卷调查集成
- 面试结果实时反馈


## 技术栈
React + WebRTC + Agora/TRTC + Node.js + Fastify
