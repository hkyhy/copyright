# 数字人视频自动生成系统

## 软件信息
- 名称：数字人视频自动生成系统
- 版本：V1.0
- 描述：基于蝉镜API的数字人视频自动生成系统，支持文本转数字人视频、异步任务处理和实时进度监控

## 主要功能
- 文本转数字人视频
- 集成蝉镜数字人API
- 多数字人形象支持
- 异步任务处理
- 实时进度监控
- 自动回调处理
- 智能错误重试
- AccessToken自动管理


## 技术栈
Node.js + Fastify + 蝉镜API + React + Ant Design
