# 中国软件著作权自动化申请指南

本项目已集成软件著作权自动化申请工具，可以快速生成申请所需的全部材料。

## 快速开始

### 方式1：使用Python脚本

```bash
# 为当前项目生成软著材料
python3 scripts/generate_copyright_docs.py . ./copyright-output

# 查看生成的文件
ls -la copyright-output/
```

### 方式2：在Kiro中使用

直接对Kiro说：
- "帮我生成软著申请材料"
- "我要申请软件著作权"
- "生成版权申请文档"

Kiro会自动分析项目并生成所有材料。

## 生成的文件

```
copyright-output/
├── 软件著作权登记申请表.md    # 申请表，需要填写著作权人信息
├── 源代码文档.md              # 前30页+后30页源代码
├── 用户手册.md                # 软件使用说明
└── 项目信息.json              # 项目统计信息
```

## 常见问题

### Q: 代码行数不够怎么办？
A: 工具会自动选择核心代码文件。如果行数不足，可以：
- 增加代码注释
- 包含更多源文件
- 确保没有排除重要的代码目录

### Q: 如何修改生成的文档？
A: 生成的是Markdown格式，可以：
1. 直接编辑.md文件
2. 转换为Word后编辑
3. 重新运行脚本生成

### Q: 软件名称和版本号如何确定？
A: 工具会自动从package.json读取：
- name字段 → 软件名称
- version字段 → 版本号

如需修改，请编辑package.json后重新生成。

### Q: 申请需要多长时间？
A: 
- 普通申请：30-40个工作日
- 加急申请：10-20个工作日（需额外费用）

### Q: 申请费用是多少？
A: 
- 个人申请：免费
- 企业申请：300元（官方费用）
- 代理费用：根据代理机构而定
