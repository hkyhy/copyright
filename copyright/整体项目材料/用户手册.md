## 四、操作说明

### 4.1 管理员操作

#### 4.1.1 系统初始化
1. 访问系统初始化页面：http://localhost:3040/admin/initAdmin
2. 注册管理员账户
3. 配置系统基本信息

#### 4.1.2 内容管理
1. 登录管理后台
2. 进入"内容管理"模块
3. 点击"新增内容"或"AI生成内容"
4. 填写内容信息或输入需求描述
5. 保存内容

#### 4.1.3 数字媒体生成
1. 在内容列表中选择项目
2. 点击"生成数字媒体"按钮
3. 选择媒体形式
4. 确认生成，系统自动处理
5. 查看生成进度和结果

#### 4.1.4 业务项目创建
1. 进入"项目管理"模块
2. 点击"创建项目"
3. 填写项目信息（名称、描述、项目类型）
4. 选择项目内容
5. 配置业务流程
6. 保存并开启项目

#### 4.1.5 邀请成员
1. 进入项目详情页
2. 点击"邀请成员"
3. 输入成员信息（姓名、手机号、邮箱）
4. 批量导入或单个添加
5. 发送邀请链接

### 4.2 用户操作

#### 4.2.1 接受项目邀请
1. 收到邀请链接或邀请码
2. 点击链接或输入邀请码
3. 填写个人信息
4. 进入项目流程

#### 4.2.2 设备检测
1. 允许浏览器访问摄像头和麦克风
2. 测试设备是否正常
3. 调整设备设置
4. 确认进入系统

#### 4.2.3 视频通信
1. 阅读操作引导
2. 查看任务和准备时间
3. 开始视频通话
4. 完成任务
5. 提交结果

#### 4.2.4 查看结果
1. 完成任务后查看结果页
2. 填写反馈问卷（如有）
3. 等待结果通知

## 七、常见问题

### 7.1 视频无法录制
- 检查浏览器权限设置
- 确认摄像头和麦克风正常工作
- 尝试刷新页面重新授权

### 7.2 数字媒体生成失败
- 检查API配置是否正确
- 确认API额度是否充足
- 查看任务日志获取详细错误信息

### 7.3 内容生成失败
- 检查AI API配置
- 确认需求描述是否清晰
- 检查网络连接
