## 🔧 Markdown转Word方法

### 方法1：使用Pandoc（推荐）
```bash
# 安装pandoc
brew install pandoc

# 转换文档
pandoc 源代码文档.md -o 源代码文档.docx
pandoc 用户手册.md -o 用户手册.docx
pandoc 软件著作权登记申请表.md -o 软件著作权登记申请表.docx
```

### 方法2：在线工具
- 访问 https://www.markdowntoword.com/
- 上传Markdown文件
- 下载Word文档

### 方法3：使用VS Code
- 安装"Markdown PDF"插件
- 右键Markdown文件
- 选择"Markdown PDF: Export (docx)"
