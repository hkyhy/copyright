## 🚀 快速开始

### 方式1：使用npm命令（推荐）
```bash
# 交互式生成
npm run copyright

# 生成整体项目材料
npm run copyright:full

# 生成指定AI模块材料
npm run copyright:modules ai-question-generator
```

### 方式2：直接使用脚本
```bash
# 生成整体项目
python3 copyright/scripts/generate_copyright_docs.py . ./copyright/整体项目材料

# 生成AI模块
python3 copyright/scripts/generate_module_copyright.py digital-video-generator .
```

### 方式3：对Kiro说
- "帮我生成软著申请材料"
- "生成数字人模块的软著材料"
